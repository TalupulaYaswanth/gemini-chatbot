## ✨ Features

- 🎨 **Premium dark UI** — glassmorphism, animated background orbs, smooth transitions
- 💬 **Multi-turn conversations** — full chat history with context memory
- 📁 **Multiple conversations** — sidebar with session management
- 🤖 **Model selector** — Gemini 2.0 Flash, 1.5 Flash, 1.5 Pro
- 📝 **Markdown rendering** — code blocks with syntax highlighting & copy button
- 🔒 **Secure API key** — stored in localStorage, never sent anywhere except Google
- 📱 **Responsive design** — works on desktop and mobile
- 💡 **Suggestion cards** — quick-start prompts on the welcome screen
- ⌨️ **Smart textarea** — auto-resize, Shift+Enter for newlines

## 🚀 Quick Start

### Option 1: Open Locally
1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Enter your Gemini API key in the sidebar
4. Start chatting!

### Option 2: GitHub Pages (Live Demo)
Visit: **[your-github-username.github.io/gemini-chatbot](https://your-github-username.github.io/gemini-chatbot)**

---

## 🔑 Getting a Gemini API Key

1. Go to [Google AI Studio](https://aistudio.google.com)
2. Sign in with your Google account
3. Click **"Get API key"** → **"Create API key"**
4. Copy the key (starts with `AIza...`)
5. Paste it in the chatbot sidebar and click **Save Key**

> **Note:** The API key is stored only in your browser's `localStorage`. It is never uploaded to any server.
